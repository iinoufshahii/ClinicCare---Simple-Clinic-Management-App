# ClinicCare – Simple Clinic Management App

**ClinicCare** is an intuitive, mobile-ready application built with Kotlin for streamlined clinic management. The app features user-friendly interfaces and robust functionality designed to improve the operations and record-keeping for small clinics, private practices, and healthcare professionals.


---

## Features

- **Patient Management**  
  Easily add, update, and view patient profiles, medical histories, and visit logs.

- **Appointment Scheduling**  
  Schedule and track appointments, view upcoming schedules, and manage cancellations or rescheduling with ease.

- **Medical Records**  
  Secure storage of consultation notes, prescriptions, diagnosis details, and treatment plans.

- **Billing & Invoicing**  
  Simple invoice generation and payment tracking for patient visits and procedures.

- **Notifications**  
  In-app alerts and reminders for appointments, follow-ups, and clinic events.

- **User Roles & Access**  
  Support for multiple user roles (admins, doctors, receptionists) with customizable permissions.

- **Data Security**  
  All patient and clinic data are handled securely following standard healthcare data protection guidelines.

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/iinoufshahii/ClinicCare---Simple-Clinic-Management-App.git
   ```
2. **Open in Android Studio:**
   - Launch Android Studio and select `Open an existing project`.
   - Navigate to the cloned folder and open it.
3. **Build the Project:**
   - Let Gradle sync complete.
   - Build and run the app on your preferred emulator or device.

> **Requirements:**  
> - Android Studio Flamingo or newer  
> - JDK 11+  
> - Android device/emulator running API level 22+

---

## Usage

- **Register New Patients:**  
  Navigate to the Patients section, fill out details, and save.

- **Schedule Appointments:**  
  Use the Appointment module to select patient, time, and physician.

- **Record Treatments:**  
  Log diagnosis and treatments per patient visit.

- **Generate Invoices:**  
  Utilize Billing section after visits to create and manage invoices.

## Technologies Used

- **Kotlin:** 100% of the codebase
- **Android Jetpack components**
- **Room Database** for offline data storage
- **Material Design** libraries
